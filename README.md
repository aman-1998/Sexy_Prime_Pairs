Sexy_Prime_Pairs:

In mathematics, Sexy Primes are prime numbers that differ from each other by six. 
For example, the numbers 5 and 11 are both sexy primes, because they differ by 6.

Sample Input:

2 40

Sample Output:

7

Explanation:

(5, 11) 

(7, 13) 

(11, 17) 

(13, 19)

(17, 23) 

(23, 29) 

(31, 37)
